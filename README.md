#  Snake Game

Um clássico jogo da Cobra desenvolvido em **Java**, criado com o objetivo de praticar e consolidar fundamentos de lógica de programação e orientação a objetos.

---



Este projeto é uma reimplementação do famoso jogo da Cobra (Snake), onde o jogador controla uma cobra que cresce à medida que come os alimentos que aparecem na tela. O jogo termina quando a cobra colide com as paredes ou com o próprio corpo.

O projeto foi desenvolvido como exercício pessoal para afiar habilidades de programação, aplicando conceitos como:

- Lógica de movimentação e colisão
- Manipulação de eventos de teclado
- Renderização gráfica com Java Swing
- Estrutura e organização de código orientado a objetos

---

##  Como Executar

### Pré-requisitos

- [JDK 8+](https://www.oracle.com/java/technologies/downloads/) instalado
- Uma IDE como [IntelliJ IDEA](https://www.jetbrains.com/idea/) (recomendada), Eclipse ou VS Code com extensão Java

### Passos

1. Clone o repositório:
   ```bash
   git clone https://github.com/gbzadaf/snake-game.git
   ```

2. Abra o projeto na sua IDE preferida.

3. Navegue até `src/application` e execute a classe principal (`Main.java` ou equivalente).


---

##  Como Jogar

| Tecla | Ação |
|-------|------|
| ↑ | Mover para cima |
| ↓ | Mover para baixo |
| ← | Mover para a esquerda |
| → | Mover para a direita |

- Come os alimentos que aparecem na tela para crescer e aumentar a pontuação.
- Evite colidir com as paredes ou com o próprio corpo.
- O jogo termina ao ocorrer uma colisão.
